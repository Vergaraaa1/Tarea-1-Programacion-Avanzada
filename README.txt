README

El código presentado tiene una clase llamada CaminosPCB la que implementa dos soluciones para contar caminos en una grilla. Una solución con programación dinámica y otra usando arboles binarios, además utiliza un decorador para medir el tiempo de ejecución de cada método. Finalmente al correr este código este debería entregar un grafico mostrando los tiempos de ejecución de cada método para poder compararlos. El código tiene por predeterminado contar los caminos desde una grilla 2x2 hasta una 10x10 manteniéndose simétrico, como ejemplo para comparar los tiempos.
